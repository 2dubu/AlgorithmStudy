## 문제 설명

자연수 N이 주어지면, N의 각 자릿수의 합을 구해서 return 하는 solution 함수를 만들어 주세요.

예를들어 N = 123이면 1 + 2 + 3 = 6을 return 하면 됩니다.

</br>

## 제한 사항

- n의 범위: 100,000,000 이하의 자연수

</br>

## 예시

**입출력 예**

|  N   | answer |
| :--: | :----: |
| 123  |   6    |
| 987  |   24   |

</br>

## Solution

```swift
func solution(_ n:Int) -> Int {
    return String(n).map{ Int($0.description)! }.reduce(0){ $0 + $1 }
}
```
